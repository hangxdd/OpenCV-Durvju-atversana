About user control system

This system is created for a face recognition algorithm that uses OpenCV and deepface libraries and reference images to recognize the user seen in the camera, to then accordingly output a text in the terminal.

This system only allows access to a authorised Django Administrator to add new users, edit or delete existing ones.

Also, there is the option to navigate to the system's Django admin panel and the Amazon's S3 file server where the authorised users pictures are stored.

This system is made using Django, Tailwind and Amazon's library boto3.
